      Parameters

      newBitmap  Bitmap
      The Bitmap object to be used as the canvas surface. This object holds the pixel data for the canvas.
      newGraphics  Graphics
      The Graphics object to be used for rendering on the canvas. This object provides drawing capabilities like shapes, text, and images.

      Remarks

      This method initializes or reinitializes the internal canvas by setting the Bitmap and Graphics objects. After setting the canvas, it clears the drawing surface with a white background to provide a fresh start. If you need a specific background color, you can modify the method to accept a color parameter or change the default behavior.

      Example

      The following example demonstrates how to initialize a canvas with a 500x500 Bitmap:
      C#
      Bitmap bitmap = new Bitmap(500, 500);
Graphics graphics = Graphics.FromImage(bitmap);
SetCanvas(bitmap, graphics);

      Exceptions

      CanvasException Thrown if either newBitmap or newGraphics is null.
