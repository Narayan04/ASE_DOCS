      Remarks

      This constructor sets the canvas size to the default values defined by the constants `XSIDE` and `YSIDE`. It also initializes the pen size to 1 and sets the pen color to black (RGB: 0, 0, 0).

      Exceptions

      CanvasException Thrown if the canvas size cannot be set properly or if invalid values are provided for pen size or color.
