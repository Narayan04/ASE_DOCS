      Clear Class

      Represents a command that clears the Canvas. The command does not take any parameters.

      Remarks

      The Clear class is used to clear the contents of the Canvas. Since no parameters are required for this command, the CheckParameters(String) method is left empty.
      • Clears all elements on the Canvas when executed.
      • Does not throw any exceptions by itself, as it has no parameter validation.
